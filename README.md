LIBRARY MANAGEMENT SYSTEM 


PROJECT PURPOSE 
* The project is a Library Management system for my university project made completely with python.
* The system is fully capable of managing books, members and all the borrowingss returns of books through a menu driven system. 
* The application uses Object Oriented Programming principles and CSV files are being used to store the data which help in loading the data
when the program is started and saving the data when the program is exited.


PROJECT FEATURES
* Add new books
* Add new members
* Borrow Books
* Return Books
* Display all Members
* Display all Books
* Display all Loans
* Automatic generation of Book and Member id
* Prevents Duplicate Member and Book id
* Load Data from CSV files 
* Save Data into CSV files
* Input Validation and Exception Handling


STRUCTURE
* main.py - The Main Menu and User Interface
* library.py - Library Class and Operations
* book.py - Handles the Book Class
* member.py - Member Class
* loan.py - Loan Class
* storage.py - CSV file Handling functions
* books.csv - Books storage file
* members.csv - Storag File for Members
* Loans.csv - Loans storage File


TECHNOLOGY USED 
* Python 3
* CSV File Storage
* Object-Oriented Programming (OOP)


EXAMPLE USAGE 
1. Open the Project Folder.
2. Run the main.py.
3. Select any of the options from the 8 given ones.
4. if you put anything above 8 it would give you an error.
5. For example you choosed option 1. then you will be asked to enter the book details like name, author and year of publication
6. Once you enter the book the book is added succesfully
7. And at last you exit the program it saves the operations automatically
8. You can now look into the CSV files that the new book is added
